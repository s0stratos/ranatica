
# TAG GUIDE

## PREFIXED VARIABLE TAGS

- +l+XXX lexeme for the word
- +r+XXX root for the word (+r+DÌC [benedictīs])
- +p+XXX prefix for the word

## MORPHOLOGY TAGS

- +ÀC  contains the AC suffix [audācissimē]
- +ÀLIS
- +ÀRE verb in ARE usually derived from noun [aritetat]
- +ÀRIUS contains the ARIUS adjectival suffix [asināriam]
- +QUE contains the particle QUE [assiduēque]
- +ÀTUS contains the ATUS suffix [barbātōs]
- +COMPARATIVE is a comparative form [audācius]
- +COMPOUND compound (= 2 stems or more in one word) [armiger]
- +CULUS contains CULUS as a diminutive suffix
- +DIMINUTIVE is a diminutive form
- +DUM contains the particle DUM [ascrībedum]
- +È is an adverb in E [audācissimē]
- +ÈLA
- +ELLUS contains ELLUS as a diminutive suffix [asellum]
- +EUS
- +IA  contains the IA suffix [audāciam]
- +ICUS contains the ICUS suffix [barbaricās]
- +IDUS
- +IENSIS contains the IENSIS suffix [athēniēnsis]
- +IMPERATIVE is an imperative form [aspice, aspicitō]
- +ISSARE contains the ISSARE suffix [atticissat, badīzās]
- +ITER adverb in ITER [assimiliter]
- +IV
- +LENTUS
- +MENTUM contains MENTUM as a suffix
- +NE contains the particle NE [audītin]
- +ND is a ND form [assentandumst]
- +NT is a NT form
- +OC contains the OC suffix [atrōcem]
- +ON
- +ST contains the ST form of ESSE [audiendumst]
- +SUPERL is a superlative form
- +TAT contains the TAT suffix [auctōritātem]
- +TIA  contains the TIA suffix [avāritia]
- +TION contains the TION suffix [ascēnsiōnem, assentātiunculās]
- +TOR contains the TOR human agent suffix [auctōritās]
- +TRIX contains the TRIX human agent suffix [assentātrīx]
- +TIM adverb in TIM [articulātim]
- +ULUS diminutives in ULUS
- +VOCATIVE is a vocative form [barbare]

## SEMANTIC TAGS

- +ANIMAL animal name, transferred from animal name, verb or adjective from animal name [aritetat]
- +ARCHITECT
- +BODY body parts, body features [barbā, barbātum]
- +CLOTHES is a name for clothes [baxeae]
- +EQUIPMENT is a man-made object (tools, weapons, clothes, furniture, machines)
- +FAMILY any kin relationship
- +FOOD
- +GEOGRAPHY contains a named place name [asiam, athēniēnsī]
- +HEALTH
- +INSULT word used as an insult
- +JOB any occupation [ballistāriō]
- +NAME person's name (real or fictional)
- +PLACE any type of place (different from +GEOGRAPHY) [balineae]
- +PLANT plant name, plant matter, plant food [assulātim]
- +TIME
