NOTES
-------
foot and half foot divisions follow syllable divisions
| fer{t adu}|
au|{t homi}nem |

noteworthy are the following clusters
- un|s
- g|n
- x|s   |cit max|sumam |

geminated consonants are always written with () if unmetrical :
{ade}|(ō) hocc in|dignum |

elided vowels are counted as part of the half foot when written.
this should be always done but especially important for monosyllabic words
because they can contribute to the weight
